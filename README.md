# Course Recommendation Chatbot

## Overview

The Course Recommendation Chatbot is an intelligent assistant designed to help students and learners discover technical courses tailored to their interests and proficiency levels. Powered by web scraping, natural language understanding, and personalized recommendation algorithms, this chatbot offers a seamless and engaging learning experience.

## Key Features

- **Web Scraping**: The chatbot collects real-time course data from a curated website, ensuring up-to-date recommendations.

- **Personalization**: Users receive course suggestions customized to their interests and proficiency levels, enhancing their learning journey.

- **Interactive Interface**: The chatbot offers a user-friendly, conversational interface for intuitive interactions.

- **GPT-2 Integration**: Natural language understanding is powered by the GPT-2 language model, providing context-aware responses.

- **Exit Functionality**: Users can exit the chatbot gracefully at any time by typing 'exit.'

## Getting Started

### Prerequisites

- Python (3.7 or higher)
- Required Python packages: `requests`, `beautifulsoup4`, `transformers`

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/course-recommendation-chatbot.git

2. Navigate to the project directory:
   cd course-recommendation-chatbot

3. Install the required Python packages:
    pip install -r requirements.txt

   Usage
Start the chatbot by running the course_recommendation_chatbot.py script:
python course_recommendation_chatbot.py

- Interact with the chatbot by typing questions or commands.
- Receive personalized course recommendations and filter courses as needed.
- Exit the chatbot at any time by typing 'exit.'

Customization and Extensibility
The Course Recommendation Chatbot is designed to be highly customizable and extensible. You can:

Fine-tune the recommendation algorithm for improved accuracy.
Implement additional features, such as user authentication or error handling.
Integrate the chatbot with a database for enhanced data management.
Deploy the chatbot on a cloud platform for scalability and accessibility.
